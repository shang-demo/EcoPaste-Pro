# 📱 安卓手机 Tasker 数据同步指南

本指南将指导您通过导入 **XML 配置文件** 的方式，快速打通 PC 端 EcoPaste 与安卓端 Tasker 的双向剪贴板同步。

---

## 📥 场景一：从 PC 推送到手机 (接收数据)

在手机上建立监听服务，自动接收并处理来自电脑的剪贴板内容。

### 1. 导入配置
打开 Tasker，长按顶部的 **【配置文件 (Profiles)】** 标签，选择 **【导入配置】**，导入 [To_Phone.prf.xml](To_Phone.prf.xml) 文件。

### 2. 配置电脑端 Webhook
在 EcoPaste 的 `偏好设置 -> 数据同步 -> 推送数据` 中，将推送通道切换为 **「自定义 Webhook」**：

*   **请求地址 (URL)**:
    ```bash
    http://192.168.10.11:4123/EcoPaste
    ```
    > [!IMPORTANT]
    > **寻址提示**：请将 `192.168.10.11` 替换为您手机真实的局域网 IP。默认端口为 `4123`，若修改端口，请同步更改 Tasker 中 `To_Phone` 触发事件的端口号。

*   **Payload 模板**:
    ```json
    {
      "content": "{{剪贴板内容}}",
      "type": "{{类型标签}}"
    }
    ```

### 3. 功能表现
配置生效后，手机接收到推送时会弹出 Toast 提示：
*   **文本类内容**: 自动写入手机系统剪贴板，可直接粘贴。
*   **图片类内容**: 自动下载并保存至手机目录：`/Download/EcoPaste/`。

> [!TIP]
> **进阶定制**：若需修改图片保存位置，请在 Tasker 中进入 `任务 -> 写入手机`，找到第 5 步的 `HTTP Request` 动作，修改文件下载路径参数即可。

---

## 📤 场景二：从手机推送至 PC (发送数据)

触发动作后，自动将手机当前剪贴板内容发送至 EcoPaste。

### 1. 导入配置
在 Tasker 主界面，长按顶部 **【配置文件】**，导入 [To_EcoPaste.prf.xml](To_EcoPaste.prf.xml) 文件。

### 2. 获取接口地址
1. 在电脑端 EcoPaste 的 `偏好设置 -> 数据同步 -> 接收数据` 中，开启 **【启用自动接收服务】**。
2. 复制面板中的接口地址，通常格式为：
    ```bash
    http://192.168.10.5:41234/api/write
    ```

### 3. 配置 Tasker 目标 URL
在 Tasker 中进入 **To_EcoPaste** 任务，找到 `HTTP Request` 动作，将上述接口地址粘贴到 **URL** 字段并保存。

### 4. 触发与权限避坑
> [!CAUTION]
> **权限限制说明**：由于安卓 10+ 隐私限制，应用在后台无法直接监听剪贴板变化。因此，原生的“剪贴板已变更”触发器在锁屏或后台状态下大概率会失效。

**推荐方案**：进入 `To_EcoPaste` 配置文件，根据习惯将触发条件修改为 **主动触发**：
*   **传感器**：例如“摇晃设备 (Shaking)”。
*   **硬件按键**：例如“长按音量键”。
*   **交互辅助**：桌面快捷方式、悬浮球或下拉快开面板。

---

通过上述配置，即可实现稳定且高效的跨平台剪贴板同步体验。
